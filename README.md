## Overview

Radar ODS is an open-access decision-support platform that measures and visualises SDG maturity in coastal municipalities of the Maranhão Equatorial Margin, Brazil. It applies the **ISM-SDG methodology** (Cunha et al., 2025, *Marine Policy*) — originally developed for Brazilian ports — to four pilot municipalities: **São Luís, Santo Amaro, Humberto de Campos, and Tutóia**.

The platform translates official open data into a composite index (scale 1–5) across five SDGs, enabling municipal managers to diagnose gaps, simulate policy interventions, and identify accessible climate-finance opportunities.

---

## Key Features

**Area 1 — Diagnostic (ISM Module)**
- ISM-SDG score per municipality and per SDG, on a 1–5 maturity scale
- Interactive radar chart visualising SDG performance
- Gap analysis against the 2030 Agenda target (ISM = 4.0)
- Comparative ranking across all four pilot municipalities

**Area 2 — Decision Support**
- Real-time simulation: adjust SDG indicators via sliders and observe the impact on the ISM score
- Funding radar: identifies which climate-finance calls (BNDES, BID, Fundo Amazônia, FNDR) are unlocked at each ISM level
- AI-generated, municipality-specific action recommendations calibrated to current scores and open calls
- Field diagnostic questionnaire to complement and validate official data

**Additional Modules**
- Collaborative network: cross-municipality exchange of best practices
- Reference panel: methodology documentation and full source list

## Methodology

The ISM-SDG index is built on the framework published in:

> Cunha, D.R., et al. (2025). *SDG Attributes: A Sustainability Assessment Framework for Brazilian Ports.* **Marine Policy**. UFMA.

Scores are computed across five SDG dimensions:

| SDG | Theme |
|-----|-------|
| SDG 7 | Affordable and Clean Energy |
| SDG 11 | Sustainable Cities and Communities |
| SDG 13 | Climate Action |
| SDG 14 | Life Below Water |
| SDG 17 | Partnerships for the Goals |

Maturity levels:

| Level | Label | Description |
|-------|-------|-------------|
| 1 | Initial | No implementation |
| 2 | Emerging | Isolated actions |
| 3 | Developing | Partial implementation |
| 4 | Advanced | Consolidated |
| 5 | Reference | Leadership and replication |

## Data Sources

All data are open-access official Brazilian sources (2022–2024):

| Source | Content |
|--------|---------|
| IBGE Censo 2022 | Energy access, sanitation, solid waste, housing |
| SINISA 2024 | Water and sewage coverage by municipality |
| IBGE MUNIC 2023 | Municipal plans, early-warning systems, open data |
| ANEEL SIGEL 2024 | Distributed solar generation |
| CEMADEN 2024 | Climate risk, hydrological alerts |
| IBAMA / MapBiomas 2022 | Land cover, mangrove preservation |
| MPA 2023 | Artisanal fisheries management |
| ANA 2023 | Water quality, hydrological monitoring |
| CGU 2023 | Municipal transparency index |
| TCU / SEFAZ-MA 2024 | Climate-finance access |

## Pilot Municipalities

| Municipality | Highlight | Priority SDGs |
|-------------|-----------|---------------|
| São Luís | State capital; full SINISA data coverage | SDG 11, SDG 17 |
| Santo Amaro | No registered solar generation (ANEEL 2024) | SDG 7, SDG 11 |
| Humberto de Campos | Strongest baseline in SDG 14 (fisheries, mangroves) | SDG 14, SDG 17 |
| Tutóia | Best overall ISM (3.28); active climate action plan | SDG 13, SDG 11 |

## Climate Finance Integration

The platform maps accessible funding calls based on the municipality's ISM score:

| ISM Level | Programme | Agency | Focus SDGs |
|-----------|-----------|--------|-----------|
| ≥ 2.0 | Mangrove Conservation | Fundo Amazônia / BNDES | SDG 14 |
| ≥ 2.5 | Sustainable Municipal Infrastructure | BNDES Clima | SDG 7, SDG 13 |
| ≥ 2.5 | Urban Infrastructure — Northeast | FNDR | SDG 11 |
| ≥ 2.8 | Sustainable Coastal Cities | IDB | SDG 11, SDG 14 |
| ≥ 3.5 | Sustainable Cities Award / Open Data | Fed. Gov. / UNESCO | All SDGs |
